Role Selection

Student Dashboard

This is the Student Dashboard, where students can navigate between different sections like "My Chats," "My Requests," and "Tutor Suggestions."

Student Navigation Menu

This is the Student Dashboard with the side menu open. From here, students can easily navigate their messages, check their profile, search for tutors, view their requests, or log out. The main dashboard in the background provides options to manage requests, explore tutor suggestions, and start a chat with a tutor when needed.

Tutor Search Window

This is the Student Dashboard with a pop-up where students can enter the subject they want to study and search for a tutor. They can either start the search or cancel.

Suggested Tutors

This is the Tutor Suggestions page, where students can browse a list of available tutors. Each tutor's profile displays their name, email, subjects they teach, and ratings if available. Students can view a tutor's profile to learn more or choose a tutor to connect with.

Request Session

Sign Up Form

Allowed to register new students into the system by collecting their comprehensive personal, guardian, and educational information.

Student Login

Empty Student Dashboard

Populated Student Dashboard

Student Portal Menu
